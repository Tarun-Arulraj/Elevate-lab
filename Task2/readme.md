Task-2 — Phishing Email Analysis (Flipkart Sample)

This folder contains all analysis files generated from a real-world–style phishing email impersonating Flipkart.

Files Included
- `phishing_email.txt` — Raw HTML email body
- `email-header.txt` — Raw extracted headers
- `sender-analysis.md` — Sender identity validation
- `header-analysis.md` — Authentication check interpretation
- `Link-Analysis.md` — Link extraction and phishing URL analysis
- `Phishing_Analysis_Report.pdf` — Final summarized report

Objective
Analyze a phishing attempt by checking:
1. Sender identity & spoofing indicators  
2. Email headers (SPF, DKIM, DMARC)  
3. Embedded malicious URLs  
4. Content & social engineering tactics  

This demonstrates the ability to identify and report phishing threats in a structured, professional manner.
